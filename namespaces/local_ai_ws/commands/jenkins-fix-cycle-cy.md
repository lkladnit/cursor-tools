# Jenkins Fix Cycle (Cypress): Fetch CI Failures, Analyze, Fix, Validate

Renamed from `/jenkins-fix-cypress`.

See the kubevirt-ui-cursor command for the full detailed workflow: `kubevirt-ui-cursor/.cursor/commands/jenkins-fix-cycle-cy.md`.
